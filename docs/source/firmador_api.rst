================
Firmador API
================

Descripción General
-------------------

Esta sección documenta la API de Firmador_API, un servicio de firma digital de documentos.

**Repositorio del Proyecto:** https://codeberg.org/firmador/firmador_api

.. warning::
   El código proporcionado en este repositorio es una base de entrada y requiere configuraciones adicionales para poder funcionar correctamente y ponerse en producción. Este no pretende ser un código de libre acceso plug-and-play. Se recomienda revisar la documentación de despliegue y realizar las configuraciones de seguridad necesarias antes de usar en entornos productivos.

   **Limitación de Swagger en ReadTheDocs:** Por motivos de seguridad, los navegadores bloquean la interfaz interactiva de Swagger cuando se ejecuta desde ReadTheDocs, ya que intenta acceder a localhost.
