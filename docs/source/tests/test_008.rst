Descripción del Test
--------------------

En la ventana de Configuración:
Se debe modificar cada parametro de la firma en esta ventana.
Luego, se debe aplicar sin guardar.
Firmar un documento PDF y verificar el cambio de parametro.
Luego se debe guardar y cerrar la ventana.
y se debe volver a abrir la ventana de configuración para verificar que el cambio se ha aplicado.
y una vez mas firmar el documento y verificar el cambio.

**Tipos de archivo soportados:**

- **Documentos:** PDF, DOCX, PPTX, XLSX, ODT, ODP, ODS, ODG
- **XML:** XML, XMLA
- **Otros formatos de texto:** TXT, CSV, MD, JSON, YAML
- **Imágenes:** PNG, JPG, BMP, TIFF, SVG
- **Multimedia:** MP3, WAV, MP4
- **Comprimidos:** ZIP, RAR, ASCIS
- **Otros:** QDS

Resultado esperado
------------------

EL archivo debe ser firmado correctamente y la firma debe mostrarse con los parametros modificados.
