Descripción del Test
--------------------

En la ventana de directorios
seleccionar un directorio y verificar que se despliega la información de los documentos y subdirectorios que contiene.
luego presionar el botón Firmar verificar que cada documento se firma correctamente.

- **Multimedia:** MP3, WAV, MP4
